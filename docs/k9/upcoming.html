RNX OS - Features and Upcoming features in RNX
---------

	Index - Table



Features : 
	
  1	Run Linux Desktop on android 
  2	Both CLI(command line interface) and GUI(grafical user interface) supported
  3	3D support
  4 	Internet supported
  5	Run Linux as an Android App
  6	Support startup programs (rnx_uwax services)
  7	Touch control
  8	Awnto Virtual Keyboard
  9	Physical Keyboard support
  10	Physical Mouse support
  11	Audio Support
  12	Mass Storage support usb,memory_card [available at folder "/awnto/shared/storage" ]
	

Upcoming :
	
  1	Shared Clipboard
  2	Android Camera support
  3	Linux Media Player listed in android notification panel
  1	Support for rooted devicez
  2	Support for 32bit arm devices
  3	Support for Android devices lower than android 7.0
	  

What is not yet supported : 

  1	Mounting (on non-rooted devices)
  2	Camera 
  3	Hardware Support
