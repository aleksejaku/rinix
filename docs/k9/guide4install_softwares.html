RNX OS - Installing Softwares
---------

	Index - Table



to install softwares like firefox , vlc
	only root user can install/update softwares
	command 'su' is use to switch to root


su


switch to root user then


apt update


update sources repository database then


for chrome


apt install chromium


for vlc


apt install vlc


for other softwares 
just search like 
example : "how to install vlc on debian"
remember our OS is arm64 == aarch64 not x86 == i386 , x86_64 == amd64



installing softwares from external sources 
example : Visual Code Studio



dowload Visual code in .deb arm64 Link
then login to root user by command 'su'
then 'apt install /sdcard/Download/code_*.deb'
all done


run "code --no-sandbox" in GUI terminal as user
to start visual code
