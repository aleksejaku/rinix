General Softwares : 




FireFox
	Web Browser firefox has audio bug in RNX
	run below command as root user to install
apt install firefox-esr


Chromium
	Web Browser
	run below command as root user to install
apt install chromium


VLC
	Video Player
	run below command as root user to install
apt install vlc


Rhythmbox
	Audio Player
	run below command as root user to install
apt install rhythmbox


UGet
	Files downloader for GUI
	run below command as root user to install
apt install uget


Transmission
	Torrent Software : you can speed limit upload/download speeds and much more
	run below command as root user to install
apt install transmission


GEdit
	Text Editer : you can customise it for programming , change its theme , much more
	run below command as root user to install
apt install gedit


Atril
	Documents Viewer 
	run below command as root user to install
apt install atril


Okular
	Documents Viewer 
	run below command as root user to install
apt install okular


Libre Office
	Office Software : Alternative to Microsoft office
	run below command as root user to install
apt install libreoffice

XArchiver
	Compress and Decompress files/folders
	run below command as root user to install
apt install xarchiver


OnBoard
	On Screen Keyboard : you can set auto hide/show in settings , change theme/layout , much more
	run below command as root user to install
apt install onboard


Gimp
	Photo Editor : graphics editor used for image manipulation and image editing, free-form drawing 
	run below command as root user to install
apt install gimp
